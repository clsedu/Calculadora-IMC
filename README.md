# Calculadora de IMC

Projeto em React + Typescript fazendo uma calculadora de IMC.

Projeto desenvolvido durante o curso [B7Web](https://b7web.com.br).



### Instalação
- `npm install`

### Para rodar
- `npm start`

### Finalidade
- demonstrar o aprendizado de ReactJs em um aplicação front-end, simples porém com várias técnicas usados em typescript.
